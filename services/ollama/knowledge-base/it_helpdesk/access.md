# IT Access

## Password reset

- Self-service portal at company login; link expires 15 minutes.
- Five failed attempts lock account 30 minutes automatically.

## VPN connection

- Install corporate VPN from software catalog.
- MFA required; split tunnel disabled by policy.

## New employee setup

- Laptop provisioning 2 business days after HR approval.
- Email and Teams on first login with temporary password.

## Shared drives

- Request folder access through IT portal; manager approval required.
- Quarterly access reviews; idle 90 days removed.

## Account lockout

- Unlock via helpdesk after identity verification.
- Never share passwords or MFA codes including with IT staff.

## Mobile device enrollment

- MDM enroll for company email on phone demo policy.
- Remote wipe if device lost — report immediately.

## Guest and contractor access

- Sponsor submits form; access expires on end date automatically.

## Help for access questions not listed

- See guide.md; have employee ID ready for unlock calls.
