# IT Helpdesk Guide

## General help for any tech issue

- Confirm username and device asset tag if assigned.
- First steps: reboot device, check VPN if remote, verify network connection.
- Collect screenshot of error for ticket attachment.

## Password and access

- Self-service portal link expires 15 minutes.
- Account lockout clears after 30 minutes or helpdesk unlock after ID check.
- MFA required for VPN and email externally.

## Software and installs

- Standard apps via Software Center; custom apps need ticket approval.
- Run updates during maintenance window when possible.

## Hardware

- Printer: confirm network, reinstall driver, test other user if shared.
- Laptop: dock, external monitor, audio — check device manager after updates.

## Network and VPN

- CORP-NET enterprise login; guest Wi-Fi not for work systems.
- VPN split tunnel off; disconnect and reconnect if stale session.

## Help when your issue is not listed

- Open ticket with category closest match; include steps already tried.
- Critical outage: call helpdesk emergency line on intranet demo page.
