# IT Helpdesk Overview

## What this line helps with

- Any workplace IT issue: passwords, login, email, VPN, Wi-Fi, software, installs, updates, printers, laptops, phones, and account access.
- Step-by-step troubleshooting for common and uncommon tech problems in demo corporate environment.
- Guidance when your issue is not in our lists using standard helpdesk procedures.

## How to ask

- Describe error messages, device type, and whether you are on-site or remote.
- Say if issue started after an update or new software install.

## Questions not in our lists

- See guide.md for systematic troubleshoot: reboot, network, permissions, reinstall.
