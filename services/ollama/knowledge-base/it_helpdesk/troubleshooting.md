# IT Troubleshooting

## Email not syncing

- Quit Outlook and restart; check VPN if off-site.
- Clear cached credentials if prompted repeatedly.

## Printer offline

- Confirm printer on office network not guest Wi-Fi.
- Remove and re-add printer in Devices and Printers.

## Laptop slow performance

- Restart weekly; keep 15 percent disk space free.
- Run Software Center updates during lunch.

## Wi-Fi issues

- Forget and reconnect CORP-NET; WPA2 enterprise login.
- Report persistent drops with floor and time to helpdesk.

## Software install requests

- Submit ticket; standard apps approved within 1 business day.
- Unauthorized software blocked; director exception required.

## Blue screen or crash

- Note stop code if visible; submit ticket with screenshot.
- Boot safe mode instructions in intranet demo KB article 1001.

## Teams or Zoom AV issues

- Check default mic and camera in app settings.
- Close other apps using camera; reboot if device not detected.

## Browser and web apps

- Clear cache; try private window to test extension conflict.
- Supported browsers: latest Chrome and Edge demo policy.

## Help for problems not listed

- Use guide.md systematic steps; escalate with ticket number if open.
