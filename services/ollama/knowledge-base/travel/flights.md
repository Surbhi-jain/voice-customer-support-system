# Travel Flights

## Booking changes

- Flex fares: same-day changes often free; basic fares 75 dollar fee.
- Name corrections up to 3 characters free once per ticket.

## Flight delays

- Over 3 hours: meal vouchers at participating airports demo policy.
- Over 5 hours: automatic rebook to next available on flex fares.

## Seat selection

- Standard seats free at 24-hour check-in.
- Extra legroom 45 dollars each way domestic demo.

## Cancellations

- Refundable tickets refund in 7 days.
- Non-refundable: airline credit minus 100 dollar fee.

## Unaccompanied minors

- Ages 8 to 17 direct flights only; 150 dollars each way.

## Missed connections

- Airline rebooks if single ticket and delay airline-caused on flex fare.
- Separate tickets: self-transfer risk; no automatic protection.

## Upgrades

- Bid upgrades in app 24 hours before flight where offered.
- Counter upgrades at gate if space available; card payment.

## Help for flight questions not listed

- Use guide.md; have confirmation code ready for changes.
