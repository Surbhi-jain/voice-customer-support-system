# Hotel Guest Care Guide

## General help for any stay issue

- Greet the issue calmly; confirm guest name and reservation if relevant.
- For timing questions, state standard policy first, then exceptions (loyalty, availability).
- Escalate to front desk supervisor for safety, harassment, or unauthorized charges.

## Reservations and changes

- Date changes depend on rate type; flexible rates often free 48 hours before arrival.
- Room type upgrades at check-in if inventory allows; quote supplement before confirming.

## During the stay

- Housekeeping requests: towels, linens, cleaning — usually within 20 to 30 minutes.
- Noise complaints: offer room move if available; log issue for management.
- Wi-Fi issues: password on key sleeve; reboot in-room router if provided.

## Check-out and billing

- Remind standard check-out time; offer late checkout if policy allows.
- Email folio within 24 hours; billing disputes within 14 days per policy.

## Help when your question is not listed

- Use the closest FAQ topic (check-in, amenities, parking, pets).
- If still unclear, advise guest to call front desk extension 0 for live assistance in demo mode.
