# Retail Support Guide

## General help for any order issue

- Confirm order number and email on account.
- Check tracking status before escalating delay.
- Never ask for full card number; demo mode only.

## Shipping and delivery problems

- If tracking stuck 5 plus days, open delay inquiry.
- Wrong address: change only before packed status.
- Lost package: start trace then replacement or refund per policy.

## Returns and refunds

- Start return in order history; prepaid label provided.
- Refund timing depends on payment method and inspection.

## Product and sizing help

- Size chart on product page; exchange free for apparel within window.
- Out of stock: notify when back or suggest similar SKU in demo catalog.

## Promotions and payment

- Promo codes at checkout; one code per order unless stated.
- Failed payment retries once in 24 hours.

## Help when your issue is not listed

- Map issue to orders.md or returns.md section.
- Complex cases: advise submitting help form with photos for damage claims.
