# Retail Returns

## Return window

- Most items returnable within 30 days of delivery.
- Final sale items marked non-returnable on product page.

## Return process

- Start return from order history; print prepaid label.
- Drop at authorized courier; refund after warehouse receipt.

## Refund timing

- Original payment 5 to 7 business days after inspection.
- Store credit within 24 hours if selected at return creation.

## Damaged or wrong items

- Report within 48 hours with photos via help form.
- Replacement ships free once verified.

## Exchanges

- Size exchanges free for apparel in window.
- Color exchange if in stock; otherwise refund.

## International returns

- Demo US only; return label US addresses.
- Customs fees non-refundable on international demo orders.

## Help for return questions not listed

- See guide.md; keep photos and order number for damage claims.
